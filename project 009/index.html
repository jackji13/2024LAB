Scrape Elements with Full Attributes and CSS

This tool allows you to scrape specific HTML elements from any website, including their full attributes, CSS styles, and screenshots. Simply enter the URL of the website you want to analyze, select the type of element you're interested in (e.g., input, button, h1, hr), and click "Scrape". You'll see the results displayed below, along with screenshots of each element. This tool is powered by a custom API that leverages Puppeteer to fetch and process elements from the target website.
